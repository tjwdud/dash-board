## :runner: 실행 방법

1. 레포지토리를 클론하고자 하는 디렉토리에서 아래 명령어를 수행

   ```
   git clone https://github.com/tjwdud/coding-test.git
   ```

<br>

2. 디렉토리 이동

   ```
   cd coding-test
   ```
   

<br>

3. 클론한 디렉토리에서 module 설치

   ```
   npm install
   ```


<br>

4. 서버 실행

   ```
   npm run server
   ```

<br>

5. client 실행

   ```
   npm run start
   ```
